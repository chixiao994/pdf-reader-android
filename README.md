# PDF阅读器 Android应用

基于Kivy开发的PDF阅读器，支持夜间模式、手势翻页、阅读进度记忆等功能。

## 📱 功能特性
- 📖 PDF文件阅读
- 🌙 夜间/日间模式切换
- 👆 手势翻页和点击控制
- 📚 阅读进度自动记忆
- 🖼️ 页面缓存，快速加载
- 📐 半边页阅读模式
- 🎯 简洁直观的界面


## 🔧 本地构建
```bash
# 安装buildozer
pip install buildozer

# 克隆仓库
git clone https://github.com/chixiao994/pdf-reader-android.git
cd pdf-reader-android

# 构建APK
buildozer android debug
